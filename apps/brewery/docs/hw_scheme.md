Пивоварня строится по аналогии с Шпайдель БрауМастер.
    == Базовый комплект
    - нагревательный элемент низкой плотности 3 КВт
    - Корзина с солодом, через которую циркулирует сусло
    - две помпы (MP-15RM 220V, BSP 1/2", SS голова) для прокачки сусла через корзину. 
    - термодатчик ds18b20
    - контроллер управления - STM32F103C6T6/C8T6
    - текстовый экран 4*20
    - блок кнопок 4 шт.
    == Дополнения
    - помпа для перекачки сусла из варочной емкости в ферментер
    - датчики протока для каждой помпы ()
    - safety контроллер STM8F0
    - Разъем для SD-карты.



**УЗО тип A** — тип AC не видит постоянную составляющую утечки, которая возможна через SSR. Тип A ловит оба вида.

**Два DS18B20** — один для управления, второй на STM8 как независимый канал. Если основной контроллер завис, safety читает температуру сам и рубит ТЭН.

**Hardware interlock между STM32 и STM8** — UART для heartbeat недостаточен: если STM32 завис в цикле и продолжает слать heartbeat, STM8 ничего не знает. Нужна отдельная GPIO-линия, которую STM32 тоггает каждые N мс. Если тоггл пропал — STM8 рубит реле напрямую, минуя STM32.

**Обратный клапан только на рециркуляционных помпах** — на перекачочной (котёл → ферментёр) он не нужен, там процесс разовый.

**Переходники NPT→BSP** — без них вся сантехника будет на герметике, что для пищевого применения нежелательно.
